📌 How to Use

  1. Select a module from the cards above
  2. Upload your SAP CSV file
  3. Upload your SFDC CSV file
  4. Select the key column for matching
  5. Click "Compare" to see the results
  6. Analyze the comparison report and export if needed

💡 Tips

  • Ensure your CSV files have matching column names unless using Dispatch Details, which supports field mapping
  • The key column is crucial for accurate matching
  • Use the summary cards to understand data quality
  • Export results for further analysis or sharing
  • Green highlights indicate matches, red shows mismatches
